# Komari Glassmorphism

个人二开定制版本，基于 `komari-theme-Glassmorphism` 调整，用于自用的 Komari Monitor 主题。

本仓库主要保留个人需要的界面、布局和主题配置改动，不作为通用主题文档维护。更多原始功能说明请参考上游项目。

![preview](/docs/preview.png)

## 使用

从本仓库 [Releases](https://github.com/Su-cyber-art/komari-theme-Glassmorphism/releases) 下载最新的 `komari-theme-Glassmorphism-build-*.zip`，然后在 Komari Monitor 后台的主题管理中上传使用。

## 开发

```bash
bun install
bun run dev
bun run lint
bun run build
```

构建后会在项目根目录生成：

```text
komari-theme-Glassmorphism-build-<commit>.zip
```
